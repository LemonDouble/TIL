### 1. Java의 naming convention

1. 첫 번째 글자는 문자, $, _여야 하고 숫자로 시작할 수 없다.
2. 영어 대소문자는 구분된다
3. 예약어 사용 불가
4. 변수 길이 제한은 없음

### 2. 리터럴 (literal)

- 정수 리터럴
    - 소수점이 없는 정수 리터럴은 10진수로 간주한다. (0 , 75, -100)
    - 0으로 시작되는 리터럴은 8진수로 간주한다 (02, -04)
    - 0x, 0X로 시작하고 0~9 숫자, abcdef로 구성된 리터럴은 16진수로 간주한다. (0x5, 0xA, 0xAC08)
- 실수 리터럴
    - 소숫점이 있는 리터럴 : 10진수 실수 ( 0.25, -3.14)
    - 대문자 또는 소문자 E가 있는 리터럴은 10진수 지수와 가수 (5E7 = 5 * 10^7, 0.12e-5 = 0.12*10^-5)
- 문자 리터럴
    - 작은 따옴표 : 문자 하나
    - 큰 따옴표 : 문자열

### 3. Scope

- 기본적으로 변수는 블록 스코프

### 4. 데이터 타입 : Primitive(원시) 타입

- 논리 리터럴을 직접 저장하는 타입
- 정수
    - byte (1byte), char(2byte), short(2byte), int(4byte), long(8byte)
- 실수
    - flaot (4byte), double(8byte)
- 논리
    - boolean (1byte)

- MSB(Most Significant Bit)는 정수값의 부호를 결정, 따라서 실제 값을 저장하는 부분은 1bit 적다
- char은 UNICODE(2byte)로 처리
- long은 리터럴 끝에 L 붙여줘야 한다.
- float : 부호 1bit, 지수 8bit, 가수 23bit
- double : 부호 1bit, 지수 11bit, 가수 52bit

### 5. 데이터 타입 : Reference(참조) 타입

- Object의 주소를 저장하는 타입

- 배열 타입
- 열거 타입
- 클래스
- 인터페이스
